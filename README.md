# ReportHub

> Dashboard framework with Angular.js and Materialize.

#### Requirements

- VirtualBox v5.0.10
- Vagrant v1.7.4


## Getting started

> Vagrant installs server environment

---
	cd ~/ngmapp
	vagrant up
	
#### Installs the following

- nginx v1.46
- git v1.9.1
- node v0.12
- npm v3.5.2
- pm2 v0.15.10
- sailsjs v0.11.2
- sails-postgresql
- bower v1.6.5
- grunt v0.4.x
- grunt-cli v0.1.13
- mongodb v3.0.7
- postgres v9.3 + postgis v2.1
